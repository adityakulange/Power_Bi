# Samsung-Data-Analysis (Interactive Dashboard creation using Power Bi)

## Project Overview

This project focuses on analyzing Samsung’s supply chain, logistics, inventory, shipment, and customer performance using Power BI.
The dashboard provides end-to-end visibility into revenue, profit, supplier efficiency, inventory levels, shipment delays, and customer-wise sales performance to support data-driven decision making.

## Dataset Used
- <a href="https://github.com/adityakulange/Power_Bi/tree/main/Samsung%20Project/Files">Samsung_Supply_Chain</a>

## Dashboard Sections
1. Supplier Analysis
- Supplier-wise average lead time
- Identification of suppliers with best delivery performance

2. Inventory Analysis
- Inventory stock by product name
- Inventory value and quantity overview

3. Shipment Analysis
- Total shipments and delivered quantity
- Carrier-wise shipment delays
- Shipment quantity and delay insights

4. Customer & Revenue Analysis
- Customer/platform-wise total revenue
- Identification of highest revenue-generating platforms
 - Amazon
 - Flipkart
 - Best Buy
 - Media Markt
 - Samsung Store
